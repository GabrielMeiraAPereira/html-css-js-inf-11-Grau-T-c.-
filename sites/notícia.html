Qual será o hardware necessário para rodar GTA6? Vamos descobrir!

Qual será o hardware necessário para rodar GTA 6? aposto que você fez essa pergunta, supondo que você seja um jogador de PC. O lance que não sabemos, mas, podemos estimar com base no hardware atual e no game Red Dead Redemption 2. Pra isso vamos mirar 60FPS 1080p com um mix entre High e Ultra sem FSR ou DLSS. Aqui vai a minha recomendação.

  • Ryzen 5 5600 ou coisa melhor
  • 16GB RAM mínimo do mínimo
  • RTX 4060 ou coisa melhor
  • 200-300GB de SSD para armazenar o game
